IMAGES FOLDER
=============

Place your guitar photos here for easy organization.

RECOMMENDED FOLDER STRUCTURE:
----------------------------

images/
  ├── guitars/
  │   ├── guitar1/
  │   │   ├── main.jpg
  │   │   ├── front.jpg
  │   │   ├── back.jpg
  │   │   ├── detail1.jpg
  │   │   └── detail2.jpg
  │   ├── guitar2/
  │   │   └── ...
  │   └── ...
  ├── hero/
  │   └── hero-image.jpg
  ├── about/
  │   └── workshop.jpg
  └── README.txt (this file)

RECOMMENDED IMAGE SIZES:
-----------------------

Guitar Cards (thumbnail): 800 x 600px (4:3 ratio)
Hero Image: 1600 x 1000px (16:10 ratio)
Detail Page Main: 1200 x 900px (4:3 ratio)
Detail Page Thumbnails: 400 x 400px (1:1 ratio)
About Page: 1000 x 1000px (1:1 ratio)

FILE FORMAT:
-----------
JPG or PNG recommended
Optimize for web (compress to 100-300kb per image)

TIPS:
----
- Use descriptive filenames (e.g., "quilted-maple-front.jpg")
- Keep files under 500kb for fast loading
- Use consistent lighting and backgrounds
- Shoot at high resolution, then resize for web
